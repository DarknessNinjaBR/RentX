# 🚗 RentX API

## ⚙️ Installing
Clone the project
```
https://github.com/DarknessNinjaBR/RentX.git
```
inside the project folder, run
```
yarn
```
create a PostgreSQL database called:
```
rentx
```
after that, run the migrations
```
yarn typeorm migration:run
```
start the application
```
yarn dev
```
Enjoy!
